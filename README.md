# App for make pixel art patterns

<img width="1142" alt="pixel-art-pattern-screenshot" src="/src/img/pixel-art-pattern-screenshot.png">

[Go to pixel art patterns site](https://pixel-art-pattern.com/)

## Install

```sh
make install
```

## Start

```sh
make start
```

## Build

```sh
make build
```

## Tests

```sh
make test
```

## Lint

```sh
make lint
```
